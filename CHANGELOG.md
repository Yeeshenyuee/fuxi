# Changelog

## [1.0.0] - 2026-06-27

首个正式版。核心能力:

- **学科自适应教学**:先判类型(A 理工逻辑 / B 文科记忆 / C 语言 / D 操作实务 / E 混合),按考点路由到对应方法文件;标杆讲解样例(few-shot)保证输出质量。
- **开课前对齐(启动铁律)**:先 intake(资料确认 + 学习者画像菜单,含开放自述)、停下等回答;资料只能由用户提供,绝不自行联网搜资料。
- **记得住·考得出**:每章自动产 Q/A 闪卡(附 Anki 导入块)、间隔重复复盘(当天/1天/3天/1周)、错题本(错因五分类+重练过关制)、考期倒计时逐日计划(考前2天冲刺模式)、考前一页纸。
- **不跑题**:教材类长资料先建章→页码索引,按章只读对应页;具体数字/年份须可回指资料出处,未覆盖处明确标注(防编造硬规则)。
- **图文并重**:9 种 SVG 图型配方;导出文件自包含化(类→硬编码属性),浏览器可直接打开。
- **掌握度与进度**:mastered 硬判定(正确率≥80% 且无待重练)、SVG 进度看板、出卷向薄弱点加权、新会话一键续学恢复摘要。
- **跨平台**:Claude Code(Skill 机制)与 Codex(AGENTS.md 入口)均可用;抽料/出图不依赖专有工具(pdftotext / scripts/render_pdf.py / scripts/extract_docx.js / SVG 写文件)。
- 附:examples/ 效果示例、evals/ 回归与触发评测集、一键安装提示词(README)。

# 记忆工具(闪卡 · 间隔重复 · 错题本 · 倒计时日程)

"讲懂"只完成一半;这套工具负责"**记得住、考得出**"。四件套全部落盘,跨会话可续。

## 一、闪卡(每章讲完必产)

每章讲解结束后,把该章考点做成 **8~15 张 Q/A 闪卡**,写入 `闪卡/<章节>.md`(模板见 `templates/闪卡.md`):
- **一卡一点**:一张卡只考一个事实/概念/公式,不出"论述题"。
- **问题具体**:"强励倍数一般取多少?"✔;"简述强励"✘(那是简答不是闪卡)。
- **答案短**:一行内;计算卡答案=公式+一句用法。
- 每张卡带标签:`章节 | 类型(A/B/C/D) | 易错?`。
- 文件末尾附 **Anki 可导入 CSV 块**(`问题,答案,标签` 三列,逗号内容用引号包),用户可直接导入 Anki/Quizlet。

## 二、间隔重复(把"回看"变成日程,不是口号)

- 复盘节奏:**当天 → 隔1天 → 隔3天 → 隔1周**(艾宾浩斯简化版)。
- 每个"复盘点"的动作 = 抽该章闪卡让用户答 + 重练错题本里到期的题,答错的卡标 `易错`,进入更密的循环。
- 复盘点**写进倒计时日程**(见下),不是让用户自己记。
- 用户说"复盘/抽卡/考考我第X章" → 从对应闪卡文件随机抽 5~8 张,先问后答。

## 三、错题本(自测批改后自动记,考前专攻)

批改模拟卷/练习后,**每道错题自动追加**进 `错题本.md`(模板见 `templates/错题本.md`),字段:
- 题目(或摘要)| 你的答案 | **错因分类**:概念不清 / 套路没认出 / 计算失误 / 记忆缺失 / 审题偏差 | 正解要点 | 出处(章节)| 重练日期 | 状态(待重练/已过关)
- 错因分类决定补法:概念不清→回阶段3换讲法;套路没认出→补题型套路表;计算失误→自检习惯;记忆缺失→加闪卡;审题→圈关键词训练。
- 重练判定:同类题**换数字/换情境**再出一道,做对才标"已过关"。
- 用户说"过错题/错题重练" → 抽状态=待重练且到期的题。考前一天默认整本过一遍。

## 四、倒计时日程(考期一到手就生成)

intake 拿到**考试日期**后,立即生成 `倒计时计划.md`(模板见 `templates/倒计时计划.md`):
- 输入:剩余天数 × 章节数 × 每章权重(考点多/计算重=权重高)× 薄弱点 × 用户每天可用时长。
- 排法:前段=逐章首学(重章多排);中段=自测+错题;**考前2天=只做错题本+闪卡+押题清单+一页纸速记,不学新内容**。
- 每天一行:`D-N | 日期 | 任务(章节/自测/复盘点) | 预计时长 | 完成☐`。间隔重复的复盘点直接嵌进对应日期。
- 进度落后时(用户说"昨天没学")→ **重排剩余天数**,砍枝节保核心,别平均摊。
- 无考期(自学)→ 改为"每周节奏"版:学新 5 天 + 复盘 1 天 + 休息 1 天。

## 五、掌握度判定 + 进度看板(mastered 不许凭感觉标)

- **判定规则**:某章 `mastered=true` 需同时满足——①该章抽卡/自测**正确率 ≥80%**;②错题本里该章**没有"待重练"**。正确率 <60% → 标进 `weak_points`,倒计时计划里给该章加码重排。
- **进度看板**:用户问"进度/看板"→ 先给章节状态表(未讲 / 已讲 / 掌握✅ / 薄弱⚠️,含各章正确率),再出一张 SVG 横条看板(绿=掌握、黄=已讲未掌握、灰=未讲、红=薄弱)存 `图/进度看板.svg`。
- **考前一页纸**:冲刺期(D-1)生成 `一页纸.md`(模板见 `templates/一页纸.md`):核心公式 + 口诀 + 高频考点10条 + 易错5条 + **错题本教训TOP5**,全部压进一页——考前最后 1 小时只看它。

## 何时触发(总结)

| 时机 | 动作 |
|---|---|
| 每章讲完 | 产该章闪卡 → 提醒当天自测 |
| 批改完 | 错题入错题本 + 更新 weak_points |
| intake 拿到考期 / 考期变更 | 生成 / 重排倒计时日程 |
| 到复盘点 / 用户喊复盘 | 抽闪卡 + 过到期错题 |
| 考前 2 天 | 切"冲刺模式":错题本 + 闪卡 + 押题 + 生成一页纸 |
| 用户问"进度/看板" | 章节状态表 + SVG 进度看板 |

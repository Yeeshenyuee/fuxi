# C 类方法:语言

适用:英语、二外、古文、语文字词。命门是**词汇 + 语法 + 用法**,且必须有**输出练习**(光输入记不牢)。

## 词汇

- **词根词缀 / 构词法**:成组记(一个词根带出一串),比孤立背高效。
- **语境 + 例句记忆**:在句子里记词,而不是背词表;记"怎么用"不只"什么意思"。
- **高频优先**:先攻考试高频词/古文高频实词虚词。
- **词族图 / 近义辨析对比表**:同义、形近、易混并排比。

## 语法

- **规则 + 例外**:讲清主干规则,再单列"例外/特殊情况"(例外往往是考点)。
- **结构树 / 时态语态对比表**:把语法体系图示化。
- 用**例句**示范每条规则,别只讲抽象规则。

## 古文专项

- 实词一词多义、虚词用法(对比表)、特殊句式(判断/被动/倒装/省略)、词类活用——逐类建小表。
- 翻译:直译为主,踩"得分点"(关键实词、句式)。

## 输出练习(C 类不可省)

听写、完形、改错、翻译、造句、写作/作文。**输出才暴露真实掌握度**,讲完一块就练一段输出。

## 推荐图型

词族/词根树、语法结构树、时态语态对比表、近义辨析表。

## 自测方式

听写、完形、改错、翻译、限时写作;错题归类(词汇/语法/用法)针对性补。

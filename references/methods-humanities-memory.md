# B 类方法:文科记忆·框架概念

适用:历史、政治、马原/思修、人文地理、法理、文学/管理常识、各科的"纯记忆"部分。命门是**把零散点编成框架 + 高效记牢 + 考场精准提取**。死记硬背是最差策略,本文件给更省力的办法。

## 三步走:框架 → 记忆术 → 巩固

### 第1步 框架化(先骨架,后血肉)
零散知识点记不住,是因为没挂到结构上。先搭骨架:
- **分类树 / 思维导图**:把一章拆成"几大块,每块几点"。
- **时间线**:历史、发展脉络、流程类。
- **因果链**:事件的"背景→原因→过程→影响"。
- **对比表**:易混概念/人物/学派并排比(最重要的提分工具)。
讲清"**为什么是这样**"(背景逻辑)——理解了的东西比孤立事实好记十倍。

### 第2步 记忆术库(确实要硬背时才用,挑合适的)

| 记忆术 | 适合 | 怎么用 |
|---|---|---|
| **口诀 / 顺口溜** | 一串并列要点 | 编成押韵短句(如朝代歌) |
| **首字母 / 取字法** | 几点要点 | 取每点一个字串成好记的词/句 |
| **谐音法** | 数字、外文名、年份 | 把难记的音转成熟悉词(如圆周率谐音句) |
| **故事串联法** | 一长串无逻辑点 | 把各点编进一个**越荒诞越好**的小故事 |
| **记忆宫殿 / 位置法** | 大量有序条目 | 把每点"放"在熟悉房间/路线的固定位置 |
| **数字编码** | 年份、数据 | 数字→形象(2→鸭、9→球)再编故事 |
| **关键词锚定** | 大段论述 | 每段抓 3 个关键词,记词带出整段 |

讲解时:**主动帮用户编一个**(如直接给一句口诀/一个故事),而不是只说"你背一下"。

### 第3步 巩固:主动回忆 + 间隔重复
- **主动回忆**:合上书,用框架**自己复述/默写**一遍,卡壳处就是没记牢的——重点回看。
- **间隔重复**:把易忘点列成清单,按"当天→隔1天→隔3天→隔1周"回看(艾宾浩斯节奏)。
- **默写自测**:对填空/名词解释,直接出题让用户默写,再对答案。

## 答题技巧(文科考场)

- **踩点答题**:简答/论述按"要点"给分,**点全比展开重要**。教用户先列要点骨架再扩写。
- **关键词优先**:每个要点先写出判分关键词(术语),再补一句解释。
- 名词解释模板:是什么(定义)+ 特征/作用 + 一句背景或例子。

## 推荐图型(详见 diagram-recipes.md)

思维导图/框架树、时间线、对比表、因果链图、分类图。

## 自测方式

默写填空、名词解释、简答(列要点)、对比辨析、时间线复原;以**合上书主动回忆**为主,别只是重读。
